## 7. APIs y Endpoints Principales

### 7.1 Autenticación `/api/auth/`

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| POST | `/api/auth/register` | Registro de usuarios |
| POST | `/api/auth/login` | Login |
| POST | `/api/auth/logout` | Logout |
| GET | `/api/auth/me` | Usuario actual |
| POST | `/api/auth/refresh` | Refresh token |

### 7.2 Productos y Categorías

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET | `/api/productos` | Listar productos |
| GET | `/api/categorias` | Listar categorías |
| POST | `/api/admin/productos` | Crear producto (admin) |
| PUT | `/api/admin/productos/[id]` | Editar producto (admin) |

### 7.3 Órdenes y Canasta

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET/POST | `/api/ordenes` | Órdenes del cliente |
| GET/POST/PUT/DELETE | `/api/canasta` | Gestión de canasta |
| PATCH | `/api/admin/ordenes` | Actualizar estado (admin) |

### 7.4 Pagos

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| POST | `/api/pagos/create` | Crear sesión de pago |
| POST | `/api/pagos/webhook` | Webhook de Wompi |

WOMPI_CURRENCY=COP
