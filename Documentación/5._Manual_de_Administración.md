## 3. Gestión de Productos

### 3.1 Lista de Productos

**URL:** `/admin/productos`

Aquí verás todos los productos del menú con:
- Imagen miniatura
- Nombre
- Categoría
- Precio
- Estado (disponible/no disponible)
- Acciones

### 3.2 Filtros y Búsqueda

| Filtro | Uso |
|--------|-----|
| **Buscar** | Por nombre de producto |
| **Categoría** | Filtrar por categoría |
| **Estado** | Disponibles / No disponibles |
| **Todos** | Ver todos |

### 3.3 Crear Nuevo Producto

1. Click en **"Nuevo Producto"** o **"+"**
2. Completa el formulario:

#### Campos del Formulario

| Campo | Tipo | Requerido | Descripción |
|-------|------|-----------|-------------|
| **Nombre** | Texto | Sí | Nombre del producto |
| **Slug** | Texto | Sí | URL amigable (auto-generado) |
| **Categoría** | Select | Sí | Categoría a la que pertenece |
| **Descripción** | Texto | No | Descripción detallada |
| **Precio** | Número | Sí | Precio en COP |
| **Precio de Oferta** | Número | No | Precio promocional (opcional) |
| **Imagen** | Archivo | No | Foto del producto (JPG, PNG, WebP) |
| **Ingredientes** | Texto | No | Lista de ingredientes |
| **Tiempo de Preparación** | Número | No | Minutos estimados |
| **Stock** | Número | No | Cantidad disponible |
| **Disponible** | Toggle | Sí | Si se puede ordenar |
| **Destacado** | Toggle | No | Mostrar en sección especial |
| **Activo** | Toggle | Sí | Visible en el menú |

#### Recomendaciones para Imágenes

- **Formato**: JPG, PNG o WebP
- **Tamaño recomendado**: 800x600 píxeles
- **Peso máximo**: 2 MB
- **Ratio**: 4:3
- Ubicar en carpeta: `public/uploads/productos/`

### 3.4 Editar Producto

1. Click en el ícono ✏️ del producto
2. Modifica los campos deseados
3. Click en **"Guardar Cambios"**

### 3.5 Eliminar Producto

1. Click en el ícono 🗑️ del producto
2. Confirmar la eliminación
3. ⚠️ Los productos eliminados no se pueden recuperar

### 3.6 Acciones Masivas

Para realizar cambios en varios productos:
1. Selecciona los productos con el checkbox
2. Elige la acción del menú desplegable:
   - Activar / Desactivar
   - Cambiar categoría
   - Eliminar seleccionados

## 5. Gestión de Órdenes

### 5.1 Lista de Órdenes

**URL:** `/admin/ordenes`

Muestra todas las órdenes del sistema con filtros por estado.

### 5.2 Filtros de Órdenes

| Estado | Descripción | Color |
|--------|-------------|-------|
| **Todas** | Ver todas las órdenes | - |
| **Pendientes** | Sin confirmar | Gris |
| **Pagadas** | Pago confirmado | Verde |
| **Confirmadas** | Aceptadas por el restaurant | Azul |
| **Preparando** | En preparación | Naranja |
| **Listas** | Listas para entregar | Amarillo |
| **En Camino** | Siendo entregadas | Azul claro |
| **Entregadas** | Completadas | Verde |
| **Canceladas** | Canceladas | Rojo |

### 5.3 Información de cada Orden

| Campo | Descripción |
|-------|-------------|
| **Número** | Identificador único (ORD-YYYY-XXXX) |
| **Cliente** | Nombre y teléfono |
| **Fecha** | Fecha y hora del pedido |
| **Items** | Cantidad de productos |
| **Total** | Monto total |
| **Estado** | Estado actual del pedido |
| **Pago** | Estado del pago |

### 5.4 Ver Detalle de Orden

Al hacer clic en una orden verás:

#### Información del Cliente
- Nombre completo
- Teléfono (para contactar)
- Email
- Dirección de entrega
- Barrio
- Referencias

#### Detalle de la Orden
- Lista de productos
- Cantidad de cada uno
- Precio unitario
- Subtotal
- Costo de domicilio
- Descuentos aplicados
- **Total**

#### Estado del Pedido
- Estado actual
- Historial de cambios
- Método de pago

### 5.5 Actualizar Estado de Orden

Para cambiar el estado:
1. Ve al detalle de la orden
2. Usa el selector de estado o botones de acción rápida
3. El sistema registrará el cambio

#### Flujo de Estados

```
Pendiente → Pagado → Confirmado → Preparando → Listo → En Camino → Entregado
                                                    ↓
                                               Cancelado
```

#### Explicación de Estados

| Estado | Cuándo Usarlo | Acción |
|--------|--------------|--------|
| **Pendiente** | Orden creada, esperando pago | Esperar |
| **Pagado** | Pago confirmado vía Wompi | Verificar pago |
| **Confirmado** | Admin acepta la orden | Confirmar recepción |
| **Preparando** | Cocina inicia preparación | Asignar a cocina |
| **Listo** | Pedido terminado | Esperar repartidor |
| **En camino** | Repartidor salió | Asignar repartidor |
| **Entregado** | Cliente recibe pedido | Finalizar |
| **Cancelado** | Orden cancelada | Ninguna |

### 5.6 Cancelar una Orden

1. Ve al detalle de la orden
2. Click en **"Cancelar Orden"**
3. Selecciona un motivo:
   - Cliente lo solicitó
   - Producto no disponible
   - Error en el pedido
   - Otra razón
4. Si hay pago en línea, se procesará el reembolso
5. Click en **"Confirmar Cancelación"**

## 10. Respaldo y Mantenimiento

### 10.1 Respaldo de Base de Datos

Realizar respaldo regularmente:

```bash
# Comando para backup
mysqldump -u root -p krokori_db > backup_$(date +%Y%m%d).sql
```

**Frecuencia recomendada:**
- Diario: Para bases con muchas transacciones
- Semanal: Para bases con actividad moderada
- Mensual: Mínimo para bases pequeñas

### 10.2 Restauración

Para restaurar desde backup:

```bash
mysql -u root -p krokori_db < backup_20241120.sql
```

### 10.3 Logs del Sistema

El sistema registra:
- Accesos al panel admin
- Cambios en productos
- Cambios en órdenes
