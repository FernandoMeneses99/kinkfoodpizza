# Documentación de Pruebas - Krokori

## 1. Plan de Pruebas

### 1.1 Objetivo

Este documento establece el plan de pruebas para el sistema Krokori, definiendo las estrategias, tipos de pruebas, criterios de aceptación y resultados obtenidos.

### 1.2 Alcance

| Tipo | Alcance |
|------|---------|
| **Pruebas Unitarias** | Funciones de utilidad (auth, validation) |
| **Pruebas de Integración** | APIs, Base de datos |
| **Pruebas de Sistema** | Flujos completos de usuario |
| **Pruebas de Aceptación** | Funcionalidades críticas |

### 1.3 Entorno de Pruebas

| Aspecto | Desarrollo | Producción |
|--------|------------|------------|
| **URL** | localhost:3000 | tudominio.com |
| **Base de Datos** | krokori_db_test | krokori_db |
| **Wompi** | Sandbox | Producción |
| **Navegadores** | Chrome, Firefox, Safari, Edge |
